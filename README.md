# Unified Demo — Data Spine

Streamlit application connecting to BigQuery for live media reporting.

**Data source:** `res-apac-dev-skynet-au.resodigital_MelbUnified.all_clients_unified`

## Run
```bash
pip install -r requirements.txt
streamlit run app.py
```

**Auth:** Requires GCP Application Default Credentials — run `gcloud auth application-default login` first.
